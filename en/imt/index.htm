    IMT

    ERP Integration Tool

    IMTMTMigration Tool

    It is an ERP integration media that provides you to transfer the data in the ERP database to another database and also to transfer other records generated within other software (ERP, CRM, …etc.) or in Excel format into the ERP system. It is also possible to provide a modular integration with IMT. It is possible to provide an automatic transfer in certain periods defined by the task timer or manual transfer of records whatsoever wanted.

    Automatic Data Transfer
    • MT® provides automatic data transfer.
    • It is possible to transfer the records to be transferred automatically or manually in the specified periods with the task timer.
    • Can work with multiple applications with open data architecture.
    • Exported records can be printed in XML format during transfer.
    Information Records Resulting from Transfer

    Information records generated as a result of the transfer can be viewed on the IMT® management screen and the relevant log records can be stored in the database.

    Transfer History
    • Transfer history can be queried and reported on a date basis with Log Reader.
    Modular Based Integration
    • Modular based integration can be provided.
    • Secure data transfer is provided with ERP integration tools.

    Applications

     

    • In company which ERP, MES and 3rd party applications  has used  can be provide automatically integration with IMT.
    • In this way, consolidated reports can be created for KPI management which guiding strategic decisions.
    • If one of the software used in the company can not used for technical reasons, it can be provided to storage the data to be transferred.
    • In case of failure of the integration, the relevant departments are informed with email via the system.
    • Similar datas of between different softwares can be provided to entere from one location.
    • Log records can be storage in the database and reported.
    • With the integration enables the system to work online with instant data or to run at the designated times with the task scheduler.
